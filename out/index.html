My Interests

Web 3.0

I'm an active developer and investor in Blockchain and Web3 technologies that I believe will become new standards in the future. I've been testing some ETH smart contracts using the Solidity language and currently working on migrating this website to the ICP blockchain. To try and stay up to date sometimes I kick the tires with test/educational projects such as the following Node.js blockchain on Github.

React & UI Development

In more recent years I have contributed to lots of frontend web, cloud, and mobile projects including this site (see code) using wonderful libraries such as React.js and Next.js. While I have lots of experience working and managing the fullstack across my career (including web APIs and gaming consoles) I especially enjoy coding up UIs and front-ends these days.

Zero Knowledge Proofs

While still in their infancy I find zkProofs to be quite a fascinating advancement when it comes to security and privacy, allowing users to securely verify private data or other statements of truth without actually revealing the information or knowledge itself. This presents great new opportunities for authentication and verification, both on- and off-chain.

Generative AI

I've been learning more about the underlying code and technologies used to create nerual networks and large language models. While relatively new to this area (currently taking some online courses) I find the opportunities and possibilities with generative-AI to be almost un-imaginable and strongly believe these technologies will have a major impact on the world in many different ways.
